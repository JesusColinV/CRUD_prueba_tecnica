# Niveles de usuario

## Usuario nivel 1 

Este usuario es el usuario con mayor poder, puede:

* Crear
* Actualizar
* Consultar
* Eliminar

## Usuario nivel 2 

Este usuario puede:

* Actualizar
* Consultar datos de domicilio

## Usuario nivel 3

Este usuario puede:

* Consultar datos generales
